# imovie
学习Vue的第一个练习项目，一个电影单页面Web应用，实现了正在热映/即将上映影片列表、影片详细信息、城市、搜索组件、影院组件、登录界面等，数据来源于猫眼电影接口  
[点击预览项目](https://rhinycm.github.io/imovie/public/preview.pdf)

npm install
